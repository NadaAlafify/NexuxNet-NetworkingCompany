# NexusNet - Responsive Networking Company Website

A modern, fully responsive landing page for a professional networking and cybersecurity company built with **HTML and CSS only**.

##  Features

- Fully Responsive Design (Mobile ≤768px, Tablet ≤1024px, Desktop)
- Clean and Professional UI/UX
- Sticky Navigation Bar
- CSS-only Typewriter Animation in Hero Section
- Hover Effects & Smooth Transitions
- Services Section with Cards
- Portfolio Gallery with Hover Overlays
- Working Contact Form Design
- Semantic HTML5 Structure
- No frameworks (Pure HTML + CSS)

##  Responsive Behavior

- **Mobile (≤ 768px)**: Single column layout 
- **Tablet (769px - 1024px)**: 2-column layout for Services & Portfolio
- **Desktop (> 1024px)**: Full 4-column layout

##  Technologies Used

- HTML5
- CSS3 (Flexbox)
- Font Awesome Icons
- Google Fonts (via system fonts)
- Pure CSS Animations
